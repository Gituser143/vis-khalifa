About Algojourney

Algojourney is an interactive visualiser for common CS data structures and algorithms. Because the best way to learn is hands-on.

Algorithms and Data Structures are some of the most essential topics of any Computer Science course. They are often very tricky to grasp, and are taught on a whiteboard or a set example, which makes it harder to understand.

We try to address this by providing a platform that encourages the idea of explorable explanations, as popularized by Brett Victor. Users can learn data structures and algorithms by giving their own input and observing how the data is processed.
